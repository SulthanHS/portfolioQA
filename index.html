Dokumentasi Pengujian Manual & UAT

User Acceptance Testing (UAT)

User Acceptance Testing (UAT) adalah proses kritis untuk memastikan bahwa aplikasi memenuhi kebutuhan bisnis dan dapat diterima oleh pengguna akhir. Peran saya dalam UAT meliputi:

  • Merancang test scenarios berdasarkan requirement bisnis
  • Membuat dokumentasi test case yang jelas dan terstruktur
  • Memfasilitasi sesi pengujian dengan stakeholder
  • Mengorganisir feedback pengguna dan melacak status persetujuan
  • Membuat laporan kemajuan dan hasil UAT
